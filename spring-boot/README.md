# Cloud Search Sample

### Install

For build in production environment:

```
./mvnw clean install
java -jar target/sample.cloudsearch-0.0.1-SNAPSHOT.jar
```

For Front-end develop:

```
./mvnw spring-boot:run
yarn start
```

### Usage

Update Keys : Access Key, Primary(Second) Key, Secret Key
